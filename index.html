ASL Demo

GitHub Pages can only host static pages.

This repo contains a streamlit app in app.py for real-time ASL prediction. To run the demo, you must deploy/run it on a Python-capable platform (local machine, Streamlit Community Cloud, Render, etc.).

In this repository, see README.md for full setup and running instructions.

Quick start (local)

pip install -r requirements.txt
streamlit run app.py
